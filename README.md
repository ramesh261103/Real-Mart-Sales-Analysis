# 📊 Real Mart Sales Analysis (Excel Dashboard)

## 📌 Project Overview

This project analyzes sales data from a retail store using Microsoft Excel.
The dashboard provides insights into sales performance, profit, customer behavior, and city-wise trends.

---

## 🎯 Objectives

* Analyze total sales, profit, and orders
* Identify top-performing cities
* Understand monthly sales trends
* Evaluate payment methods and customer ratings

---

## 📂 Dataset

The dataset contains:

* Order details
* Product categories
* City-wise sales
* Payment methods (Cash, E-wallet, Credit Card)
* Ratings

---

## 📊 Dashboard Features

* Total Sales, Profit, Orders, and Average Rating KPIs
* Monthly Sales Trend (Line Chart)
* Sales by City (Bar Chart)
* Payment Mode Distribution (Donut Chart)
* Interactive Filters:

  * Gender
  * City
  * Product Line
  * Branch

---

## 🛠 Tools Used

* Microsoft Excel

  * Pivot Tables
  * Charts
  * Slicers
  * Data Cleaning
